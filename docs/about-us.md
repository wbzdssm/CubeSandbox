# About us

The Tencent Cloud IaaS Frontier Technology Team is part of the Tencent Cloud IaaS Product organization. We are a professional team dedicated to exploring cutting-edge technologies and achieving breakthroughs in core technical areas. Our work spans high-performance and high-availability networking, Agent Infra, KVCache, collective communication, and more. Several of our core achievements have been deployed at large commercial scale.

## Key Achievements

### Cube Sandbox

An ultra-fast, secure sandbox designed specifically for AI Agents, with industry-leading performance. It has been deployed at scale within Tencent Cloud's serverless computing and Agent sandbox platforms, serving numerous leading AI/Agent workloads both inside and outside Tencent. The project is open source.

- GitHub: [github.com/tencentcloud/CubeSandbox](https://github.com/tencentcloud/CubeSandbox)

### HARP Protocol Stack

An in-house multipath reliable transport protocol, analogous to AWS SRD. It has been implemented in Tencent Cloud's proprietary smart NICs and deployed in Tencent Cloud's storage systems, covering hundreds of thousands of compute nodes at scale.

### HARP Tunnel

Tencent Cloud's next-generation Overlay protocol, enabling sub-100ms transparent failure avoidance in the physical network with near-zero overhead. It is deployed at scale across Tencent Cloud's virtual network, covering millions of compute nodes. The related technical work has been accepted at NSDI '26:

> [*Harp: Improving VPC Network Availability via Efficient Failure Detection and Reroute in Tencent Cloud*](https://www.usenix.org/conference/nsdi26/presentation/hu-jiayu)

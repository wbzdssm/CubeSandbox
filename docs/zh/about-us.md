# 关于我们

腾讯云 IaaS 前沿技术团队隶属于腾讯云 IaaS 产品团队，是一支专注于前沿技术探索与核心技术突破的专业技术团队。团队工作范畴涵盖高性能高可用网络、Agent Infra、KVCache、集合通信等多个关键技术领域，目前已成功实现多项核心成果的大规模商业化部署。

## 核心成果

### Cube Sandbox

一款专为 AI Agent 设计的极速安全沙箱，具备行业领先的性能优势。该产品已在腾讯云内部函数计算、Agent 沙箱等核心业务中实现大规模落地，高效服务于腾讯内外众多 AI/Agent 头部业务，且已对外开源，助力行业发展。

- 开源地址：[github.com/tencentcloud/CubeSandbox](https://github.com/tencentcloud/CubeSandbox)

### HARP 协议栈

团队自主研发的多路径可靠传输协议，类似于 AWS SRD。该协议已在腾讯云自研智能网卡中完成部署实现，并成功应用于腾讯云存储系统，覆盖数十万计算节点，实现规模化落地应用。

### HARP Tunnel

作为腾讯云下一代 Overlay 协议，可实现接近零损耗的物理网络百毫秒级业务无感避障，目前已在腾讯云虚拟网络中大规模部署，覆盖百万计算节点。相关技术实践已被 NSDI 26 会议录用，对应论文题为：

> [*Harp: Improving VPC Network Availability via Efficient Failure Detection and Reroute in Tencent Cloud*](https://www.usenix.org/conference/nsdi26/presentation/hu-jiayu)
